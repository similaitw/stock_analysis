# 賴氏家族投資儀表板

這是一個不需後端、可直接在瀏覽器操作的單頁投資管理網站。

## 功能

- 家族帳戶持股管理
- 自動計算總市值、成本、損益與報酬率
- 顯示核心 ETF、成長 ETF、高股息、海外 ETF、個股、現金與黃金配置
- 編輯目標配置與每月定期投入
- 退休前 10 年複利試算
- 匯出與匯入 JSON 備份
- 自動檢查現金比率與個股比率
- 內建賴氏家族投資憲法

## 使用方式

1. 直接開啟 `index.html`。
2. 在「資產管理」更新股數、均價及現價。
3. 按「儲存」，資料會存入瀏覽器 localStorage。
4. 定期按「匯出 JSON」備份。

## GitHub Pages

在 GitHub 儲存庫中進入：

`Settings → Pages → Build and deployment → Deploy from a branch`

選擇：

- Branch：`main`
- Folder：`/ (root)`

儲存後即可取得網站網址。

## 重要提醒

- 網站不會自動取得即時股價，請手動更新現價。
- 所有資料只存在使用中的瀏覽器；清除瀏覽器資料前務必匯出 JSON。
- 這是資產紀律管理工具，不是投資建議或獲利保證。
